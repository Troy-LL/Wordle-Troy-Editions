# Wordle - Troy Editions

Custom Wordle implementations with curated vocabularies, theme-based challenges, and extended game mechanics.

Currently featuring the **Valentine Edition**—a specialized Wordle experience with a secret Valentine mode and interactive animations!

> [!NOTE]
> This project was developed in less than an hour as a demonstration of **AI-agentic coding capabilities**. 🚀

## How to Run the Game

1.  **Open your terminal** (Command Prompt or PowerShell).
2.  **Navigate to the project folder**:
    ```powershell
    cd "p:\Troy\Code\Side Projects\Wordle-Troy-Editions"
    ```
3.  **Start the local server**:
    ```powershell
    python server.py
    ```
4.  **Play the game**:
    *   The game should open automatically in your browser.
    *   If not, open your browser and go to: [http://localhost:8000](http://localhost:8000)

## Features

*   **Standard Wordle**: Play the classic game.
*   **Valentine Mode**: Click the small heart in the bottom-left corner.
    *   **Curated Vocabulary**: Specialized words for a romantic (or tricky) challenge.
    *   **Dynamic Animations**: Winning (or losing) triggers a "Will you be my valentine?" sequence.
    *   **Interactive Response**: Type **YES** for a celebratory confetti finale or **NO** for a total eclipse of the screen.
