# Wordle - Valentine Edition

## How to Run the Game

1.  **Open your terminal** (Command Prompt or PowerShell).
2.  **Navigate to the project folder**:
    ```powershell
    cd "p:\Troy\Code\Wordle"
    ```
3.  **Start the local server**:
    ```powershell
    python server.py
    ```
4.  **Play the game**:
    *   The game should open automatically in your browser.
    *   If not, open your browser and go to: [http://localhost:8000](http://localhost:8000)

## Features

*   **Standard Wordle**: Play the classic game.
*   **Valentine Mode**: Click the small heart in the bottom-left corner.
    *   Words become Valentine-themed.
    *   Winning (or losing) triggers a "Will you be my valentine?" animation.
    *   Type **YES** for a celebration!
    *   Type **NO** for a pitch-black screen.
